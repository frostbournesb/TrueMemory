# LoCoMo Dataset

`locomo10.json` contains 10 multi-session conversations from the LoCoMo benchmark.

- **Source**: [snap-research/locomo](https://github.com/snap-research/locomo)
- **Paper**: [Evaluating Very Long-Term Conversational Memory of LLM Agents](https://arxiv.org/abs/2402.17753) (ACL 2024)
- **Size**: 10 conversations, ~300 turns each, 1,540 questions across 4 categories
- **License**: Academic/research use. See the [original repository](https://github.com/snap-research/locomo) for terms.
